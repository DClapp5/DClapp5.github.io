Vertical Nav Tutorial

Lets start in Index.html

First, create a div for your nav

This will act as a container and allow us to have a background box.

            
                <div class="nav">    
                </div>

Now, add an ordered list with all your links

            
                <div class="nav">    
                    <ol>
                        <li class="navLink"><a href="#">link 1</a></li>
                        <li class="navLink"><a href="#">link 2</a></li>
                        <li class="navLink"><a href="#">link 3</a></li>
                        <li class="navLink"><a href="#">link 4</a></li>
                        <li class="navLink"><a href="#">link 5</a></li>
                    </ol>
                </div>

Now lets jump into CSS

First create a class under "*", which will apply to everything

            
                *{
                    padding:0;
                    margin:0;
                }

Next create a "nav" class

            
                .nav {
                    width: 10%;
                    height: 100%;
                    background-color: grey;
                    float: left;
                    position: fixed
                }

Now lets create a tag which applies to any "ol" which are within the "nav" div

            
                .nav ol{
                    list-style: none;
                }

Lets create a new, further specified class

            
                .nav ol li a{
                    color: white;
                    font-size: 20px
                    padding: 15px 20px
                    text-decoration: none;
                    display: block;
                    position: relative;
                }

To finish off the nav, we need to add a slight color change when the user hovers

            
                .nav ol li a:hover {
                    color: rgb(0, 0, 0);
                    background-color: rgb(189, 189, 189)
                }

And all the nav code put together should look like this:

            
                <div class="nav">    
                    <ol>
                        <li class="navLink"><a href="#">link 1</a></li>
                        <li class="navLink"><a href="#">link 2</a></li>
                        <li class="navLink"><a href="#">link 3</a></li>
                        <li class="navLink"><a href="#">link 4</a></li>
                        <li class="navLink"><a href="#">link 5</a></li>
                    </ol>
                </div>

And all the CSS code put together should look like this:

            
                * {
                    padding: 0;
                    margin: 0;
                }
                
                .nav {
                    width: 10%;
                    height: 100%;
                    background-color: grey;
                    float: left;
                    position: fixed
                }
                
                .nav ol {
                    list-style: none;
                }
                
                .nav ol li a {
                    color: white;
                    font-size: 20px;
                    padding: 15px 20px;
                    text-decoration: none;
                    display: block;
                    position: relative;
                }
                
                .nav ol li a:hover {
                    color: rgb(0, 0, 0);
                    background-color: rgb(189, 189, 189)
                }
